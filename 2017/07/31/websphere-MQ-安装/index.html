websphere_MQ_安装

发表于 2017-07-31   |   分类于 环境—工具

安装过程

1、下载

下载websphere_MQ (https://www.ibm.com/developerworks/cn/downloads/ws/wmq/)
解压后双击Setup.exe

2、点击“启动IBM WebSphere MQ安装程序(L)”，进入如下弹出框。

3、安装准备工作完成后弹出

4、默认选择“典型”，点“下一步”。

5、点“安装”，开始安装。

6、弹出安装框。

7、 过一会就显示安装完成

8、点击“完成”自动进入如下的“准备WebSphere MQ向导”

9、点“下一步”，弹出如下弹出框，因为我是Windows 7操作系统，所以选择“否”。选择是会有点麻烦

10、点“下一步”，启动向导

11、点“完成”，即进入如下的IBM WebSphere MQ Explorer界面，在这里我们可以建队列管理器等等了。

遇到的问题

1、打开安装好的websphere MQ报错“Fail to create the Java virtual machine.”
在目录C:\Program Files (x86)\IBM\WebSphere MQ\java\jre\bin下执行java.exe

再输入path,发现是path路径有中文目录，导致MQ遍历path路径出错。

删掉中文目录重启ok
