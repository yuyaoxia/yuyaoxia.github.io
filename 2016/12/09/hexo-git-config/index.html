hexo+github——很不错的独立博客！

发表于 2016-12-09   |   分类于 环境-工具

一直想拥有一个可以按照自己的爱好布局的博客网站，只到遇到你——hexo！对于不擅长搭环境的我，在同事得帮忙下，忙活了一上午，终于初见规模。想写的东西太多，但我感觉还是先聊聊我是怎么折腾好这个环境滴……

1.购买域名

推荐阿里云、godaddy，如果企业、公司建议阿里云，自己写写博客godaddy挺好，两者价格目前差不多。

2.github账号

申请一个github账号
建存储库：与你用户名要对应，仓库名必须为『your_user_name.github.io』
作用：提供远程创库、服务器、github pages（GitHub Pages默认为每个用户分配了一个二级域名『your_user_name.github.com』或『your_user_name.github.io』，所以可以不用买域名。）

3.Node.js

下载：node-v6.9.1-x64.msi
安装：一直“下一步”即可
作用：hexo安装的依赖

4.Git客户端

下载：PortableGit-2.11.0-64-bit.7z.exe，这里我用的是便携版！
安装：一路“下一步”
配置：GIT_HOME：D:\my_net\PortableGit；path:%GIT_HOME%/bin

5.hexo

hexo是一个基于Node.js的快速、简单和强大的博客框架，支持Markdown格式，可以方便的生成有很炫主题的静态网页，方便托管在github和Heroku上。

安装：npm install -g hexo
初始化到指定目录：hexo init 或者 在指定目录hexo init
