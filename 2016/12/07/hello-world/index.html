首页文章显示行数

写文章时插入,文章会自动从插入的位置截断，也就是说在博客中只显示之前的内容，点击阅读全文之后会显示所有内容。

数学公式的默认定界符

块级公式:$$…$$和\[…\]（对于块级公式）
行内公式:$…$和\(…\)
但是，如果你的文章内容中经常出现美元符号“$”, 或者说你想将“$”用作美元符号而非行内公式的定界符，请在Hexo目录的_config.yml中添加：

mathjax2: true
而不是mathjax: true。 相应地，在需要使用数学公式的文章的front-matter中也添加mathjax2: true。

3.1.1 init

新建一个网站。如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站。

3.1.2 new

新建一篇文章。如果没有设置 layout 的话，默认使用 _config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。

3.1.4 publish

发表草稿。

3.1.5 server

启动服务器。默认情况下，访问网址为： http://localhost:4000/。

选项 简称 描述
–port -p 重设端口
–static -s 只使用静态文件
–log -l 启动日记记录，使用覆盖记录格式

3.1.6 deploy

部署网站。

选项 简称 描述
–generate -g 部署之前预先生成静态文件

该命令可以简写为：$ hexo d

3.1.7 render

渲染文件。

选项 简称 描述
–output -o 部署之前预先生成静态文件
